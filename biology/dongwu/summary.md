## 原口和后口

|        原口动物        |              后口动物              |
| :--------------------: | :--------------------------------: |
|        螺旋卵裂        |              辐射卵裂              |
|    内胚层来自端细胞    |    内胚层来自体腔(脊椎动物除外)    |
| 体腔由中胚层带开裂形成 | 体腔由体腔囊融合形成(脊椎动物除外) |
| 口来自原口，肛门后形成 |       肛门来自原口，口新形成       |

## 恒温

鸟类、兽类动物维持体温恒定。

### 意义

* 促进了各种酶的活动，提高新陈代谢水平
* 提高了快速运动的能力
* 减少了对环境的依赖
